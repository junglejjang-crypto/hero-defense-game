⚔️ 영웅 디펜스 — 게임 방법

🎮 기본 조작

  • 하단 영웅 버튼 탭 → 배치 모드
  • 맵 빈 곳 탭 → 영웅 배치
  • 같은 버튼 다시 탭 / ESC → 취소
  • 배치된 영웅 탭 → 업그레이드·판매
  • 키보드 1/2/3 → 영웅 선택 (PC)

⚡ 배속 모드 / 지옥 모드

  • 1× / 2× / 3× 버튼으로 게임 속도 조절
  • 다른 탭으로 이동해도 게임 자동 진행
  • 🔥 4·8·12·14·18라운드 — 지옥 모드 (배경 변화, 몬스터 강화)

🗺️ 몬스터 출현 루트

  • Wave 1~10: 출발① 1개 루트
  • Wave 11~20: 출발①② 2개 루트 동시 출현
  • Wave 21~: 출발①②③ 3개 루트 전면 개방
  • 비활성 루트는 🔒로 표시 — 미리 영웅 배치 가능

📷 슈퍼몬스터 이미지

  • [📷 슈퍼몬스터] 버튼으로 사진 업로드
  • 슈퍼 몬스터 외형이 해당 사진으로 변경됨
